Benchmarking

A little script helped to benchmark this. The script is available here. It takes two arguments:
  1. Name of SAS program (without .sas extension)
  2. [Optional] Number of desired replications (default=5)
Thus, the 5 replications for the SQL with index were run using the following command: 
./benchmark.ksh test-sqlset 5
The benchmark script will produce a log file called [name_of_sas_program].[number_of_replications], and print the average USER_CPU, SYSTEM_CPU and I/O counts (sum of input and output), averaged over the number of replications, to the console. Thus, the replication for the SQL with index produced a file called test-sqlset.5, and printed the following to the console: 
Avg USER_CPU   for 5 runs: 5.4208
Avg SYSTEM_CPU for 5 runs: 4.2728
Avg IO         for 5 runs: 326291
